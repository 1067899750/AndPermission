分享文件

分享私有文件

为了提高私有文件的安全性，面向 Android7.0或更高版本的应用私有目录被限制访问，尝试传递file://URI会触发FileUriExposedException，分享私有文件内容的推荐方法是使用FileProvider，它将生成content://URI。

FileProvider类位于support v4包下，因此需要依赖com.android.support:support-v4:version。AndPermission是没有依赖任何第三方包，包括Google提供的support包和compat包，因此为了避免引入第三方包造成的宝体积增大问题，AndPermission拷贝了FileProvider。

AndPermission提供了兼容Android7.0及更高版本分享文件Uri的方法：

String filePath = ...;

Uri uri = AndPermission.getFileUri(context, new File(filePath));

比如在Android7.0或者更高版本上安装Apk时：

String apkPath = ...;

Intent intent = new Intent(Intent.ACTION_INSTALL_PACKAGE);
intent.setFlags(Intent.FLAG_ACTIVITY_NEW_TASK);
intent.addFlags(Intent.FLAG_GRANT_READ_URI_PERMISSION);
Uri uri = AndPermission.getFileUri(context, new File(apkPath));
intent.setDataAndType(uri, "application/vnd.android.package-archive");
context.startActivity(intent);

AndPermission提供了更加简便的方法来帮助开发者提升体验，详情请参考在Android7.0和Android8.0上安装Apk。
